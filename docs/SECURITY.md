# Security Policy

## Reporting Vulnerabilities
If you discover a security vulnerability within BREAKPOINT, please send an e-mail to security@breakpoint-web.io. All security vulnerabilities will be promptly addressed.

Please do NOT report vulnerabilities through public GitHub issues.

## Supported Versions
Only the latest release (v2.6.0-UNIFIED) is currently supported with security updates.

## Ethical Usage
This tool is for authorized security testing only. Performance of unauthorized testing is a violation of the EULA and may be illegal.
